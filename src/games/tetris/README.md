# 담당: 테트리스

이 폴더 안의 파일만 수정하면 됩니다. 다른 팀원의 폴더나 `src/App.tsx`,
`src/gameRegistry.ts` 등은 건드리지 않아도 로비 화면에 자동으로 반영돼요.

## 파일 구성

- `Tetris.tsx` — 게임 로직 + 렌더링. 실제로 채워야 할 부분입니다.
- `Tetris.css` — 보드/블록 스타일.
- `index.ts` — 로비에 표시될 제목/설명/아이콘 정보. `inProgress: true`는
  구현이 끝나면 지워주세요.

## 이미 준비되어 있는 것

- 10x20 board 상태
- 7종 테트로미노 모양 데이터(`TETROMINOES`)와 무작위 스폰 함수(`spawnPiece`)
- `DROP_INTERVAL_MS`(기본 800ms)마다 자동으로 `moveDown`을 호출하는 타이머
- 방향키 입력을 감지하는 이벤트 리스너 (동작은 아직 비어있음)

## 해야 할 일 (`Tetris.tsx` 안의 `TODO(tetris)` 참고)

1. **`canMove`**: 지금은 항상 `true`를 반환하는 더미입니다. 후보 위치가
   보드 경계를 벗어나거나 이미 채워진 칸과 겹치면 `false`를 반환하도록 구현
2. **좌우 이동 / 소프트 드롭**: 키 입력에 따라 `col`/`row`를 바꾸고
   `canMove`로 검증 후 반영
3. **회전**: `shape` 2차원 배열을 90도 회전시키는 함수 작성 후 `canMove`로
   검증
4. **고정(lock)**: `moveDown`에서 더 이상 내려갈 수 없을 때 현재 피스를
   `board`에 합치기
5. **줄 삭제**: 가득 찬 행을 찾아 제거하고, 위쪽 행들을 한 칸씩 내리고,
   지운 줄 수에 따라 `setScore`로 점수 반영 (예: 1줄=100, 2줄=300 ...)
6. **게임 오버**: 새 피스를 스폰할 자리가 없으면 게임 오버 처리

## 로컬 실행

```bash
npm install
npm run dev
```

브라우저에서 `/tetris` 경로로 바로 이동해 확인할 수 있습니다.
