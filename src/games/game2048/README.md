# 담당: 2048

이 폴더 안의 파일만 수정하면 됩니다. 다른 팀원의 폴더나 `src/App.tsx`,
`src/gameRegistry.ts` 등은 건드리지 않아도 로비 화면에 자동으로 반영돼요.

## 파일 구성

- `Game2048.tsx` — 게임 로직 + 렌더링. 실제로 채워야 할 부분입니다.
- `Game2048.css` — 보드/타일 스타일.
- `index.ts` — 로비에 표시될 제목/설명/아이콘 정보. `inProgress: true`는
  구현이 끝나면 지워주세요 (로비에서 "준비 중" 배지가 사라집니다).

## 해야 할 일 (`Game2048.tsx` 안의 `TODO(2048)` 참고)

1. **초기화**: 4x4 보드에 타일 2개(2 또는 4)를 무작위 위치에 생성
2. **이동/병합**: 방향키 입력 시 해당 방향으로 타일을 밀고, 같은 숫자가
   만나면 하나로 합치며 점수에 더하기
3. **새 타일 생성**: 이동으로 보드에 변화가 있었을 때만 빈 칸에 새 타일 추가
4. **게임 오버 판정**: 더 이상 이동/병합이 불가능하면 종료 상태 표시
5. (선택) 승리 조건 연출: 2048 타일이 나오면 축하 메시지

점수 상태(`score`)와 최고 점수 저장(`useHighScore`)은 이미 연결되어 있으니
`setScore`로 점수만 올려주면 최고 기록은 자동으로 저장됩니다.

## 로컬 실행

```bash
npm install
npm run dev
```
